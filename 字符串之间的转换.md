## 字符串之间的转换
c++常用的几个种字符串：char*, string, cstring之间的转换
### char*与string
char\*可以直接复制给string  
```c
const char* x= "hello";   
const char y[]= "world";  
string z;    
//都可以直接赋值   
z= x;   
z= y;   
```   
string转char\*：   
```c
string str= "abc";   
char* p= str.data();//1   
char* q= str.c_str();//2最好不要这么做    
//3   
char r[20];   
str.copy(r, 3, 0);//3代表复制的数量，0代表起始位置  
*(p+3)= '\0\;   
```
